# Gemini-Bot
Features:
  Dynamic Command registration
  Gemini AI powered autoplay and prompted playlist
  Gemini Ai powered chat responses
  Youtube Support
  Configurable for multiple guild support
  Fully customizable

Creating a new class in the slashcommand or textcommand path will automatically register it as a new command when the bot is started.

The bot will respond to users who @ it with AI

Multiple commands to customize the bot for your guild
